Consultation description

We launched a public consultation on a new ‘Special educational needs and disability ( SEND ) code of practice: 0 to 25 years’ in autumn 2013. This closed on 9 December 2013 and over 700 responses were received.

To make sure that the revised code is clear and easy to understand, we have:

  • analysed the responses
  • held discussions with people and organisations representing those the code applies to

We’re now consulting on the revised draft of the code of practice and invite your views on a set of specific questions.
