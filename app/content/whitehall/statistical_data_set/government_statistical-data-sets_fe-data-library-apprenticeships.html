This statistical data set provides information on apprenticeships through a number of reports broken down by starts, achievements, sector subject area, framework and standard, geography, gender, age and others. It is provides supplementary information to the further education and skills statistical first release (SFR) .

For data on apprenticeship vacancies, see FE data library: apprenticeship vacancies .
