Information on apprenticeship vacancies.

This statistical data set provides information through a number of reports on the number of apprenticeship vacancies and traineeship opportunities for 2008/09 to 2014/15 year-to-date.

Applications stated are for those made online through Find An Apprentice, and do not include those made offline directly to employers.

‘Recruit an apprentice’, replaced Apprenticeship vacancies (Av) in August with the following changes:

  • recruitment agents will not be able to use the new service - the small number of providers that use recruitment agents will therefore have to manage their own vacancies
  • subcontractors will no longer need lead provider authorisation to post and manage vacancies
