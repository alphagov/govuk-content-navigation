Admissions schema (not including ATF)

Schema files for admissions transfer files (APT, ADT, AOT, ASL and ALT) version 11 are for use in the 2014 to 2015 admissions round.

The following are available:

  • schema and includes (XSD file types) that support the 11 series specification and a readme file to accompany the scheme
  • example files that accompany the schema: these are for admissions transfer files (ATF, APT, ADT, AOT, ASL and ALT) version 11 for use in the 2014 to 2015 admissions round (resulting in admissions to schools from September 2015 and in year admissions during the academic year 2014 to 2015: in-year co-ordination is not statutory but own admission authority schools must notify the local authority of the application and its outcome)

Note: XSD files must be stored in a separate folder from those used in earlier years as some of the files have the same names but contain different definitions.
