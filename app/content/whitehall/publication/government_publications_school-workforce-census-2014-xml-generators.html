Detail

These spreadsheets allow schools and local authorities using non-standard management information systems to convert data returns for the school workforce census 2014 into DfE ’s required XML format.

Schools and local authorities working with data relating to a school’s workforce should use the ‘School workforce census 2014: school return XML generator’ spreadsheet.

Local authorities working with data relating to the local authority’s own workforce should use the ‘School workforce census 2014: central return XML generator’ spreadsheet.

The 2 guides explain how to use the generators.
