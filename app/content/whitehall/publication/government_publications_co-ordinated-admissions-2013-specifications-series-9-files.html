Detail

‘School admissions supporting documentation for series 9 files’ defines the use and specification of these files.

‘Preferences met report specification for March 2013’ covers the file on percentages of parents obtaining a preferred secondary school in March 2013 to be sent to the Department for Education.

Admissions transfer file ( ATF ) schema

Schema files are for use in the 2012 to 2013 admissions round for offers to pupils starting school in September 2013.

The following are available:

  • ATF 9.1 schema
  • ATF 9.1 example file
  • ATF 9.1 validated XML example file
  • ATF changes log

Admissions schema (not including ATF )

Schema files for admissions transfer files (APT, ADT, AOT, ASL and ALT) version 9.1 are for use in the 2012 to 2013 admissions round.

The following are available:

  • schema and includes ( XSD file types) that support the 9.1 series specification and a readme file to accompany the schema
  • example files that accompany the schema - these are for admissions transfer files (APT, ADT, AOT, ASL and ALT) version 9.1 for use in the 2012 to 2013 admissions round (resulting in admissions to schools from September 2013 and in year admissions during the academic year 2012. In-year co-ordination is still statutory under the admissions code during this period)
  • admissions changes log

Note: XSD files must be stored in a separate folder from those used in earlier years as some of the files have the same names but contain different definitions.
