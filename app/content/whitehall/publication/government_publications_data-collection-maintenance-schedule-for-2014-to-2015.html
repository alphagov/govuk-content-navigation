Details

The maintenance windows are provisional. We will publish the confirmed dates on the data collection facility page of the Hub.

During learning aims updates, the learning aims search facility on the Hub will still be available. Your Funding Information System (FIS) installation will still be available, but will not be able to call for new data until we have finished the update process.

During the essential maintenance windows the data collection system will be unavailable and we cannot guarantee the availability of the data collections facility nor of the learning aims search on the Hub, nor of the FIS update facility.

Data collection for further education providers

Data library, individual learner record (ILR) guides, community learning, learning aim reference application (LARA).

  • Individualised Learner Record (ILR): check that data is accurate
  • Individualised Learner Record (ILR): sources of data
  • SFA: supplementary data collection
