Detail

The funding calculation is implemented as a ‘rulebase’ using a rules management product called Oracle Policy Automation (OPA), which uses a natural language approach to turn statements into computer logic.

This document is compiled into the product to produce the calculations; there is no separate set of programming code to implement the logic.
