Details

The authority proforma tool (APT) is being initially issued in mid-June 2013. This note accompanies the separate APT user instructions document which has also been prepared by the EFA.

The EFA will re-issue the APT to local authorities in mid-December 2013, with the prepopulated data updated to relate to the October 2013 school census, and the March 2013 SSDA903 dataset for the looked-after children factor. An updated version of this dataset specification will be published at the same time.
