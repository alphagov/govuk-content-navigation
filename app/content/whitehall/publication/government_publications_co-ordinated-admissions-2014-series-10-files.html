Detail

Admissions transfer file ( ATF ) schema

Schema files are for use in the 2013 to 2014 admissions round for offers to pupils starting school in September 2014.

The following are available:

  • ATF 10.1 schema
  • ATF 10.1 example file

Admissions schema (not including ATF )

Schema files for admissions transfer files (APT, ADT, AOT, ASL and ALT) version 10.1 are for use in the 2013 to 2014 admissions round.

The following are available:

  • schema and includes ( XSD file types) that support the 10.1 series specification and a readme file to accompany the schema
  • example files that accompany the schema - these are for admissions transfer files (APT, ADT, AOT, ASL and ALT) version 10.1 for use in the 2013 to 2014 admissions round (resulting in admissions to schools from September 2014 and in year admissions during the academic year 2013. In-year co-ordination is still statutory under the admissions code during this period)

Note: XSD files must be stored in a separate folder from those used in earlier years as some of the files have the same names but contain different definitions.
