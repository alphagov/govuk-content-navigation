Detail

This departmental advice is for:

  • headteachers
  • test administrators
  • key stage 2 assessment co-ordinators
  • special educational needs ( SEN ) co-ordinators

Test administrators should read the relevant guide(s) before administering modified versions of the 2014 key stage 2 national curriculum tests.

The guides expand on the information in the ‘Test administrators’ guide’ .
