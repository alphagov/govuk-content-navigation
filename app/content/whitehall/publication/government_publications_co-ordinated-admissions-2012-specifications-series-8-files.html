Details

‘School admissions supporting documentation for series 8 files’ defines the use and specification of these files.

‘Preferences met report specification for March 2012’ covers the file on percentages of parents obtaining a preferred secondary school in March 2012 to be sent to the Department for Education.

Admissions transfer file ( ATF ) schema

Schema files are for use in the 2011 to 2012 admissions round for offers to pupils starting school in September 2012.

The following are available:

  • ATF 8.1 schema
  • ATF 8.1 example file
  • ATF 8.1 validated XML example file
  • ATF changes log

Admissions schema (not including ATF )

Schema files for admissions transfer files (APT, ADT, AOT, ASL and ALT) version 8.1 are for use in the 2011 to 2012 admissions round.

Note: XSD files must be stored in a separate folder from those used in earlier years as some of the files have the same names but contain different definitions.
