Details

This guidance will be updated for the 2017 test cycle in March.

This guidance is for schools participating in the 2016 key stage 2 national curriculum tests.

It provides instructions for completing attendance registers and returning test scripts (test papers containing pupils’ answers) for marking. Headteachers are responsible for ensuring these instructions are followed correctly.
