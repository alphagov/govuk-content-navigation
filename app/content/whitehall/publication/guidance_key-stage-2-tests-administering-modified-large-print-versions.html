Guidance for teachers and teaching assistants using the modified large print (MLP) key stage 2 tests with pupils with a visual impairment.

The following general guidance will help you prepare for administering the key stage 2 tests. You should follow it to ensure that pupils using the modified large print ( MLP ) versions of the tests can access them properly and that they are not at a disadvantage. You should use this guidance together with the:

  • 2015 test administration guidance ( TAG )
  • printed instructions that accompany the test papers

The printed instructions include information about what you should do before, during and after each test. Please follow them carefully in order to avoid accusations of maladministration. The instructions aren’t on GOV.UK because they contain information which must stay confidential until the test period is over.

Timings – general advice

The timings shown on the MLP test papers are the times for the standard tests.

Pupils using the MLP tests are automatically entitled to up to 100% additional time. Use your discretion about whether to use the full amount of additional time, for example with pupils who tire quickly. You should tell the pupils what time they are allowed, including any additional time, before each test starts.

Pupils’ responses

Pupils using MLP test papers will usually answer on the test paper in the spaces provided, but they may use separate sheets of paper if required.

Make sure all test scripts (answer booklets or test papers containing pupil’s answers) have the following information written on the front cover:

  • the name of the pupil, (this must be the same as the name entered in the ‘Pupil registration’ section of the NCA tools website )
  • the name of your school
  • your school’s Department for Education ( DfE ) number
  • the name of the test paper (for example levels 3-5 mathematics Paper 1)

Any separate sheets should be labelled with the information listed above and inserted inside the relevant pupil’s test scripts. If these details have not been provided by the pupil, you should add them at the end of the test session.

English tests

Mark scheme information

The number of marks and type of question are indicated within the test to help pupils judge how much to write.

Mark scheme information boxes are printed on the back cover of the test papers to avoid confusion for pupils with a visual impairment. Tell pupils that they don’t need to write anything on the back cover of the paper.

English reading test

Make sure that pupils using the MLP versions of the tests can recognise the details in all pictorial representations and photographs. You may help pupils to identify details on all pictures, but you mustn’t help them to read any text. You may stop the clock where pupils with severe visual impairment need additional time to interpret the pictures and diagrams.

English grammar, punctuation and spelling test

Spelling tasks

The instructions for administering the MLP version of the spelling task are the same as the instructions for the standard version of the test. You will need to make a copy of the standard instructions once they have been opened.

The spelling task materials will include an MLP version of the spelling task answer sheet. Pupils may write the target spellings in a numbered list form on plain paper if they prefer this approach.

The answer lines in the spelling task answer sheet are numbered to aid pupils with a visual impairment. You should use these numbers to help pupils to locate the correct line to write on, before reading aloud the sentence including the target spelling.

Pupils mustn’t use spell checkers in the spelling task.

Level 6 Paper 1: extended task

The extended task answer booklet is longer this year. If pupils ask for additional paper they should use the space in the answer booklet first. Pupils are still only expected to produce up to 3 sides of writing.

Mathematics test

Levels 3-5 mental mathematics test

You should administer the MLP version of the mental mathematics test on a one-to-one basis using the relevant section of the ‘Mental mathematics scripts for pupils with a visual impairment’.

You should not refer to the standard version of the test CD or transcript. Instructions, question order and content may differ from the standard test.

Test materials

The test materials consist of an MLP answer booklet which contains the stimulus material from the standard version of the pupil answer sheet, together with any additional stimulus material that pupils will need to access the question.

There is practice question stimulus material and an answer line on the first page of the MLP answer booklet to help pupils familiarise themselves with the test format.

The script for administering the MLP mental mathematics test may contain additional verbal prompts to help pupils to locate specific items in the MLP answer booklet. This information is printed in italics in the modified script. You must use this on the first reading of the question, but you may ignore it on the second reading of the question.

Timings

The timings of the MLP test differ from the standard test to provide accessibility for pupils with a visual impairment. You may give pupils as much additional time as they need to familiarise themselves with any stimulus material before a test question is read. Once a question has been asked, you may allow pupils up to 100% additional time to process the language of the question and record their answer. The additional time is reflected in the modified script. You must adhere to the timings in the script strictly.

The written mathematics tests

The levels 3-5 and level 6 mathematics Paper 1 and Paper 2 contain a number of diagrams, graphs and pictures. Where necessary, we have simplified or adapted these for pupils with a visual impairment, but they might still need help locating the correct diagrams or clarifying their layout. Compared with the standard versions of the tests, some non-essential diagrams and illustrations may have been omitted from the MLP versions of the test papers. In the MLP papers, diagrams are printed within the text.

Pupils will need to draw in response to some questions. Where this is the case, pupils should use the method that they think is most appropriate. If appropriate, the area covered by the shape may be shaded or marked so that the marker can clearly read it.
