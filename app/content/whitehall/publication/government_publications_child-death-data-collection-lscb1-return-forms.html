Detail

Local safeguarding children boards ( LSCBs ) should fill in these forms with the data supplied by their CDOPs about the child death reviews they have completed.

You must use the LSCB1 form for items 1 to 28 and tables 3, 4, 15 and 18.

All other information can be entered using aggregate figures on the LSCB1 form.

We encourage all LSCBs to use the optional LSCB A1 form to return data on each individual completed child death review.

The LSCB1 data return deadline is 13 May 2016.
