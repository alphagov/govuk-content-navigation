Naming your EAS

In order for the system to recognise your EAS submissions, your EAS spreadsheet filename must be in the following format: EAS_UKPRN_yyyymmdd_hhmmss.xls (for example: EAS_ 10040375_ 20131016_ 131045.xls). This information is available in the on-line data collections (OLDC) ILR changes user guide , where you will also find advice on how to upload your EAS to the OLDC.

EAS funding

The funding that can be claimed using the EAS process falls into the following categories:

  • excess learning support
  • exceptional learning support
  • audit adjustments
  • authorised claims
  • learner support

The EAS is available for each collection period during the funding year. You must enter the funding claimed for each category; this is in addition to any funding being earned from the ILR and should represent the value being claimed for each of the categories for that month. It is not the cumulative value for the funding year. Training provider statement (TPS)

The EAS replaces the TPS that was used in 2012 to 2013. In general, the process is very similar, although the design of the input form has been changed to reflect current funding policy.
