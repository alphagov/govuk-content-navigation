Request

Can the Department provide information about the secondary school enrolment rate since 1980?

Release

Given the large time frame requested and for consistency purposes, information has been provided on pupil numbers for all school types from 1979 onwards.

The information can found in the following publications, which can be downloaded from the associated resource section of this page:

  • DfE: Schools, Pupils and their Characteristics - January 2010 Statistical First Release: Information on numbers of pupils in secondary schools from 2000-2010 can be found in Table 2a.
  • Schools in England 1999 Statistical Volume: Information on numbers of pupils in secondary schools from 1979-1999 can be found in Table 1a.

We are able to provide information on the number of pupils by age from 1998 onwards; however a consistent time series is not available. The list of publications below contains information on the number of pupils by age, which can be downloaded from the associated resource section of this page:

  • DfE: Schools, Pupils and their Characteristics - January 2010 Statistical First Release: Information on the number of pupils by age as at January 2010 can be found in Tables 1a-1d.
  • DCSF: Schools, Pupils and their Characteristics - January 2009 Statistical First Release: Information on the number of pupils by age as at January 2009 can be found in Table 1a-1d.
  • Schools in England 2000 Statistical Volume: Information on the number of pupils by age as at January 2000 can be found in Table 12.
  • Schools in England 1999 Statistical Volume: Information on the number of pupils by age as at January 1999 can be found in Table 12.
  • Schools in England 1998 Statistical Volume: Information on the number of pupils by age as at January 1998 can be found in Table 12.
