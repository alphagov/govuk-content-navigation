Details

This document (version 1.1) sets out the requirements for the SLASC 2014, which is collected from registered independent schools. The requirements for general hospital schools are not part of this specification and are defined separately.

The specification covers the expected census return to the Department for Education and the validation rules which will be applied. It should be read in conjunction with the current common basic data set ( CBDS ) definitions.
