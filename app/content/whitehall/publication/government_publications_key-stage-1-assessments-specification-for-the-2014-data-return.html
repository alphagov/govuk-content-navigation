This document provides information and guidance to users submitting data for the key stage 1 (KS1) assessments data return in 2014.

Documents

Key stage 1 assessments data return 2014: business specification

Ref: DFE-00367-2014 PDF , 439KB , 28 pages

Details

This specification is for:

  • suppliers developing software for use in management information systems
  • local authorities collecting and submitting the key stage 1 ( KS1 ) assessments data return 2014

It contains:

  • the business rationale for the return
  • changes to the 2014 return
  • assessment module descriptions
  • methods for importing data into local authorities’ information systems
  • XML formats and messages
  • validation check information

This guide must be read in conjunction with the current version of the Common basic data set ( CBDS ) .
