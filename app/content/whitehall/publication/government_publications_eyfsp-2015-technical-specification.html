Early years foundation stage profile 2015: technical specification

Ref: DFE-00103-2015 PDF , 495KB , 24 pages

Details

This specification is for:

  • suppliers of software for school management information systems ( MIS )
  • local authorities collecting and submitting data for the early years foundation stage profile ( EYFSP ) return 2015

It contains:

  • changes since the previous year’s data collection
  • what data to collect at early years foundation stage provider level and child level
  • how to import data into local authority information systems
  • how to structure the data in XML
  • how to check the data against the validation rules

This guide must be read in conjunction with the current version of the common basic data set ( CBDS ) . It defines common data items that schools use in MIS software and that we use in our data collections.
