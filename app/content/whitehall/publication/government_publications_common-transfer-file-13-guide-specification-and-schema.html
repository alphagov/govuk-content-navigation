Details

This guide is for local authorities and schools to help them use common transfer file ( CTF ) 13 to transfer pupil data. The CTF is valid for use from November 2013.

The guide includes information on:

  • the statutory obligations to use CTF when transferring pupil data
  • when to send CTFs and partial data transfers
  • how to send CTFs and partial data transfers
  • filename structure
  • receiving CTF information

The schema and specification documents are for software suppliers developing and maintaining CTF files in school and local authority management information systems.
