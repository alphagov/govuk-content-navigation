Details

This document tells you:

  • how to report infant class size data to us through the school census
  • what to do when a query relating to class size appears in COLLECT

COLLECT is our centralised data collection and management system.

More infant class size guidance is available in:

  • the ‘class information module’ section of the school census guide
  • the ‘infant class size’ section of the school admissions code
