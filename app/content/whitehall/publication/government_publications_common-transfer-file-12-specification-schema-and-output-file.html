This common transfer file ( CTF ) specification was superseded in November 2013.

It should only be used as a reference document by suppliers creating software for schools and local authorities and should be read in conjunction with the schema, example output file and test XML files.

The latest version of the CTF specification can be found on the common transfer file collection .
