Details

Sure Start children’s centres can use these logos on their external signage and promotional materials.

If you’re not a Sure Start children’s centre, but are providing Sure Start services on behalf of a children’s centre, please use the additional ‘Delivering Sure Start services’ logos.

Read statutory guidance for local authorities on running Sure Start centres .
