Technical specification for software suppliers, educators and local authorities completing returns for the key stage 2 data collection 2015.

Key stage 2 teacher assessment 2015: technical specification version V1.2

Ref: STA-00626-2014 PDF , 650KB , 32 pages

Detail

This specification will enable schools, local authorities, educational establishments outside of local authority control and those such as software suppliers working on their behalf to prepare and return the required data on key stage 2 assessments made during the 2014 to 2015 assessment cycle.

The specification covers the expected teacher assessment return to the Department for Education and the validation rules which will be applied. It should be read in conjunction with the current common basic data set definitions .
