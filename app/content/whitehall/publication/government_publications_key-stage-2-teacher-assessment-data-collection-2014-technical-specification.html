This specification will enable schools, local authorities, educational establishments outside of local authority control and those such as software suppliers working on their behalf to prepare and return the required data on key stage 2 assessments made during the 2013 to 2014 assessment cycle.

The specification covers the expected teacher assessment return to the Department for Education and the validation rules which will be applied. It should be read in conjunction with the current common basic data set definitions , which can be found on the departmentâ€™s website.
