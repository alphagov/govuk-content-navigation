School census spring and summer 2014: business and technical specification

Ref: DFE-00067-2013 PDF , 914KB , 76 pages

Details

This specification defines the scope of the school census spring and summer 2014 data collections. It contains:

  • lists of any changes to the census collection
  • data module descriptions
  • XML message structure definition
  • detailed specifications for pupil- and school-level codesets
  • validation check information

This guide must be read in conjunction with:

  • the current version of the common basic data set (CBDS).
  • the validation rules workbook
  • XML schemas
  • XSLT validation rules
