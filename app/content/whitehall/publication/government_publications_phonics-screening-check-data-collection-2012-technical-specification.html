Details

This specification defines the scope of the year 1 phonics screening check data collection 2012. It provides technical details and guidance for schools, academies (including free schools) local authorities, and software suppliers to successfully prepare and submit phonics screening check data to us.

It contains:

  • lists of any changes to the census collection
  • data module descriptions
  • XML message structure examples
  • validation rules and tables

This guide must be read in conjunction with the current version of the common basic data set (CBDS) .
