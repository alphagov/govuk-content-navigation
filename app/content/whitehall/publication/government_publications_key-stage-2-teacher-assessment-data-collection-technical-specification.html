Details

This specification will enable schools, local authorities, educational establishments outside of local authority control and software suppliers working on their behalf to prepare and return the required data on key stage 2 assessments during the 2016 to 2017 assessment cycle.

The specification covers the expected teacher assessment return to the Department for Education and the validation rules which will be applied. It should be read in conjunction with the common basic data set definitions.
