How a pupil's results in the key stage 2 tests can be changed if a pupil is affected by extremely distressing circumstances.

Document

How special consideration is calculated

HTML

Detail

If a pupil’s performance in the key stage 2 tests is affected by extremely distressing circumstances, we may change the pupil’s test level to reflect this. This guidance illustrates how special consideration is calculated and applied.

Teachers who want to apply for special consideration should read key stage 2 tests: how to apply for special consideration .
