Currently, there are around 500,000 people employed in the security industry. This number includes static and patrol guards, uniformed retail security guards, store detectives, CCTV operators, dog handlers and private investigators.

Training opportunities

There are several pathways in this sector such as:

  • static and patrol guarding
  • events security
  • door supervision
  • retail security
  • reception duties
  • CCTV operation

You’ll need a calm head to assess different situations. You’ll also need the confidence to act when necessary. You’ll also learn the importance of keeping accurate records and handling documents.

Apprenticeship pay and holidays

Types of apprenticeships

This is an intermediate level apprenticeship. You can train in roles like:

  • door supervisor
  • CCTV operator
  • security officer

Apply for an apprenticeship providing security services

You can become an apprentice in England if you’re:

  • 16 or over
  • eligible to work in England
  • not in full-time education
