Use this collection to complete your statutory requirements. You should also be familiar with the phonics screening check: administration guidance ( CAG ), key stage 1 test administration guidance ( TAG ) and key stage 2: test administration guidance ( TAG ).

We regularly write to LAs about the assessment updates that we send to schools. These assessment updates help schools to complete actions on time and administer the key stages 1 and 2 tasks, tests and checks correctly.
