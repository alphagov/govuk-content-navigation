  1. Documents

This data collection contains three documents that allow awarding organisations ( AO ) and learner registration bodies ( LRB ) to create a registration batch file containing details of learners (including their ULN). Learners can then be found and their details updated within the LRS organisation portal.

The validation tool allows you to check the information contained with your registration batch file before it is submitted.
